# TestDXGI
使用DXGI获取屏幕数据  
获取鼠标光标，处理鼠标状态DXGI_OUTDUPL_POINTER_SHAPE_TYPE_MONOCHROME  
将原始RGB数据压缩为jpeg  
创建web服务，发送jpeg图片流  
客户端使用浏览器访问http://127.0.0.1:29999/  
开发环境使用Microsoft Visual Studio Community 2019
